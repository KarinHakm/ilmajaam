# Ilmajaam

Live ilmarakendus, mis näitab praegust ilma ja tunniprognoose sinu asukoha järgi.

**[Ava rakendus](https://karinhakm.github.io/ilmajaam/)**

## Funktsioonid

- Tuvastab asukoha automaatselt IP järgi
- Näitab praegust temperatuuri, tuult, niiskust ja pilvisust
- Tunniprognoosid ülejäänud päevaks
- Uuendab andmeid automaatselt iga 5 minuti tagant

## Tehnoloogiad

- Puhas HTML + JavaScript, serverit pole vaja
- [Open-Meteo](https://open-meteo.com/) — tasuta ilma-API, ei vaja API võtit
- [ipapi.co](https://ipapi.co/) — asukoha tuvastamine IP järgi
